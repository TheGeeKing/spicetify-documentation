    Checks whether two URI:s refer to the same thing even though they might not necessarily be equal.

    These two Playlist URIs, for example, refer to the same playlist:

    Parameters

Static from

  • from(value: any): null | URI
    • Defined in globals.d.ts:649

    Parses a given object into a URI instance.

    Unlike URI.fromString, this function could receive any kind of value. If the value is already a URI instance, it is simply returned. Otherwise the value will be stringified before parsing.

    This function also does not throw an error like URI.fromString, but instead simply returns null if it can't parse the value.

    Parameters

    • value: any

      The value to parse.

    Returns null | URI

    The corresponding URI instance, or null if the passed value is not a valid value.
