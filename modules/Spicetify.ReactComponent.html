    Determins is the context menu should open or toggle when triggered

  • children: typeof ContextMenuChildren

    A child of the context menu. Should be <button>, <a>, a custom react component that forwards a ref to a <button> or <a>, or a function. If a function is passed it will be called with (isOpen, handleContextMenu, ref) as arguments.

  • menu: typeof Menu | typeof AlbumMenu | typeof PodcastShowMenu | typeof ArtistMenu | typeof PlaylistMenu

    The menu UI to render inside of the context menu.

  • Optional offset?: [number, number]

    The x and y offset distances at which the context menu should open. Relative to trigger element and position.

  • Optional placement?: "top" | "top-start" | "top-end" | "right" | "right-start" | "right-end" | "bottom" | "bottom-start" | "bottom-end" | "left" | "left-start" | "left-end"

    The preferred placement of the context menu when it opens. Relative to trigger element.

  • Optional preventScrollingWhileOpen?: boolean

    Will stop the client from scrolling while the context menu is open

  • Optional renderInline?: boolean

    Decide whether to use the global singleton context menu (rendered in ) or a new inline context menu (rendered in a sibling element to children)

  • Optional trigger?: "click" | "right-click"

    Determins what will trigger the context menu. For example, a click, or a right-click
